# Vision-Based Transfer Learning System

This project implements a vision-based transfer learning system using TensorFlow, aiming to create an optimized model for production-ready inference.

## Project Structure

- `data/`: Dataset storage and preprocessing scripts
- `models/`: Trained models and checkpoints
- `scripts/`: Training, evaluation, and conversion scripts
- `api/`: FastAPI application for inference
- `docs/`: Documentation and benchmarks
- `notebooks/`: Jupyter notebooks for development and experimentation

## Phases

1. Problem definition & dataset collection
2. Data pipeline & augmentation
3. Baseline model & transfer-learning setup
4. Fine-tuning & evaluation
5. Optimization for inference
6. Serving & API
7. On-device / lightweight deployment (optional)
8. Documentation & benchmarks

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. (Optional) Configure Python environment if needed.

## Training the Model

Run the scripts in order:

1. `python scripts/data_collection.py` - Download and prepare dataset
2. `python scripts/train_baseline.py` - Train baseline model
3. `python scripts/fine_tune.py` - Fine-tune and evaluate
4. `python scripts/optimize_model.py` - Convert to TFLite

## Running the API

Run `python api/main.py` or use the "Run API" task in VS Code.

The API will be available at http://localhost:8000

- GET `/` - Health check
- POST `/predict` - Upload an image file to get prediction

Example using curl:
```
curl -X POST "http://localhost:8000/predict" -H "accept: application/json" -H "Content-Type: multipart/form-data" -F "file=@image.jpg"
```

## Benchmarks

Run `python docs/benchmark.py` after providing test image paths.

## Deliverables

- Labeled dataset: Cats vs Dogs from TensorFlow Datasets
- Trained TF model: `models/fine_tuned_model.h5`
- TFLite model: `models/model.tflite`
- Inference API: `api/main.py`
- Benchmark report: Generated by `docs/benchmark.py`
